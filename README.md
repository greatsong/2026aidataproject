# 🌡️ 서울 기온 분석기

서울의 역사적 기온 데이터를 분석하여 특정 날짜가 평년 대비 얼마나 춥거나 더운지 알려주는 웹앱입니다.

## 주요 기능

### 1. 📊 날짜별 분석
- 원하는 날짜를 선택하면 해당 날짜의 기온이 역사적으로 얼마나 특이한지 분석
- 퍼센타일 기반 비교 (예: "이 날짜 중 하위 15%의 추위")
- 역사적 기온 변화 추이 차트
- 기온 분포 히스토그램

### 2. 🎓 수능 시험일 분석 (1994~2025)
- 역대 수능일 기온 비교
- 가장 춥거나 따뜻했던 수능일 순위
- 평년 대비 편차 분석
- 수능 한파 상세 분석

### 3. 📈 전체 데이터 탐색
- 연평균 기온 변화 추이 (온난화 추세 확인)
- 월별 평균 기온
- 역대 최고/최저 기온 기록

## 사용 방법

### 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

### Streamlit Cloud 배포
1. GitHub에 이 저장소를 push
2. [Streamlit Cloud](https://streamlit.io/cloud)에서 배포
3. 기본 데이터가 포함되어 있어 바로 사용 가능

### 추가 데이터 업로드
- 사이드바에서 기상청 형식의 CSV 파일 업로드 가능
- 같은 형식(날짜, 지점, 평균기온, 최저기온, 최고기온)이면 자동 인식

## 데이터 형식

기상청 기온 데이터 형식 (CSV, EUC-KR 인코딩):
```
날짜,지점,평균기온(℃),최저기온(℃),최고기온(℃)
1907-10-01,108,13.5,7.9,20.7
...
```

## 데이터 출처
- 기상청 기상자료개방포털 (https://data.kma.go.kr)
- 서울 관측소 (지점번호: 108)
- 기간: 1907년 10월 ~ 현재
